# Matchmover
Восстановление траектории камеры по набору изображений

### Начало работы
Для запуска Вам понадобятся:
* Python 3.6
* OpenCV
* NumPy

Настоятельно рекомендую Miniconda, хотя никто не запрещает пользоваться pip 
или собрать нужные библиотеки самостоятельно.
